  1. JDK,JRE,JVM三者之间的关系，以及JDK、JRE包含的主要结构有哪些

JDK = JRE + Java的开发工具（javac.exe,java.exe,javadoc.exe）
JRD = JVM + Java核心类库

2.为什么要配置path环境变量？如何配置？

JAVA_HOME = bin的上一层目录

4.为什么配置path环境变量？
path环境变量：windows操作系统执行命令时所要搜寻的路径
为什么要配置path:希望java的开发工具（javac.exe,java.exe)在任何的文件路径下都可以执行成功。
